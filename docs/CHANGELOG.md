# Changelog

## v2.1.2 (2026-03-03)

- 移除顶部副标题“欢迎来到 永和阳光苑”，释放移动端首屏可视空间
- 移动端积分榜默认折叠条数优化：iPhone 默认 5 条，Pro Max 默认 6 条
- 未结算房间“刷新”按钮增加请求中旋转动画，明确反馈刷新状态
- Header 面板上下留白进一步收紧，整体高度小幅降低并保持布局和谐
- 撤回历史记录卡片统一高度占位方案，恢复原始卡片行为

## v2.1.1 (2026-03-02)

- 账号设置弹窗与日期筛选弹窗标签可读性优化：
  - `新昵称`
  - `新密码`
  - `开始日期`
  - `结束日期`
  以上标签文字统一为纯黑色，提升移动端与桌面端对比度

## v2.1.0 (2026-03-02)

- 新增积分榜日期范围弹出筛选（预设 + 开始/结束日期）
- 日期默认范围为当年 `01-01` 到当天，且支持快捷区间
- 点击快捷区间/日期后可自动关闭 DatePicker 弹窗
- 优化 DatePicker 透明度与布局（移动端可读性提升）
- 修复“全部玩家”卡片与 `+2k/-2k` 控件对齐问题（输入框同一行右侧）
- 积分榜改为统计全量历史对局（不再仅限当前用户参与）
- 历史记录继续仅显示当前用户参与过的对局

## v1.2.0 (2026-03-02)

- 买入逻辑支持负数冲正（纠错/还款场景），并限制累计总买入最低为 `0`
- 优化“切换房间”按钮阴影，玻璃拟态视觉更自然
- 优化“全部玩家”卡片宽度与横向滑动显示，避免移动端卡片被裁切
- 文档全量同步到 v1.2
