# Changelog

## v2.1.0 (2026-03-02)

- 新增积分榜日期范围弹出筛选（预设 + 开始/结束日期）
- 日期默认范围为当年 `01-01` 到当天，且支持快捷区间
- 点击快捷区间/日期后可自动关闭 DatePicker 弹窗
- 优化 DatePicker 透明度与布局（移动端可读性提升）
- 修复“全部玩家”卡片与 `+2k/-2k` 控件对齐问题（输入框同一行右侧）
- 积分榜改为统计全量历史对局（不再仅限当前用户参与）
- 历史记录继续仅显示当前用户参与过的对局

## v1.2.0 (2026-03-02)

- 买入逻辑支持负数冲正（纠错/还款场景），并限制累计总买入最低为 `0`
- 优化“切换房间”按钮阴影，玻璃拟态视觉更自然
- 优化“全部玩家”卡片宽度与横向滑动显示，避免移动端卡片被裁切
- 顶部副标题改为常量 `APP_SUBTITLE`，并统一为“欢迎来到 永和阳光苑”
- 文档全量同步到 v1.2

## v1.0.0 (2026-03-01)

- 完成移动端适配与玻璃拟态 UI 升级
